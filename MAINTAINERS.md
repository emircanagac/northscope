## Maintainer Responsibilities

- Review and merge pull requests.
- Triage issues and security reports.
- Keep the Helm chart, image publishing, and README accurate.
- Protect the read-only security model.
- Document major technical decisions in issues, pull requests, or README updates.

## Becoming a Maintainer

As the project grows, regular contributors may be invited to become maintainers based on sustained high-quality contributions, respectful collaboration, and good judgment around Kubernetes production use cases.
