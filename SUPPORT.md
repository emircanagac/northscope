# Support

NorthScope is pre-beta. The best support channel is GitHub Issues.

## Where to Ask

- Use a bug report for broken topology, incorrect routing, UI problems, or install failures.
- Use a feature request for new topology cases, controller support, or workflow improvements.
- Use the security policy for vulnerabilities. Do not open public issues for security reports.

## Before Opening an Issue

Please include:

- NorthScope image tag or commit
- Kubernetes version
- Ingress controller type
- Helm values or install command
- A minimal Ingress, Service, and EndpointSlice/Pod example when possible
- Screenshot or GIF when the issue is visual

This project is not a replacement for your ingress controller, cloud load balancer, or Kubernetes support vendor.
