# Code of Conduct

NorthScope follows the Contributor Covenant Code of Conduct, version 2.1.

## Expected Behavior

Participants are expected to be respectful, constructive, inclusive, and focused on improving the project. Good faith disagreement is welcome; harassment, personal attacks, intimidation, or publishing private information are not.

## Scope

This policy applies to project spaces such as issues, pull requests, discussions, documentation, community chat, and events where someone is representing NorthScope.

## Reporting

Report Code of Conduct concerns to the project maintainer listed in [MAINTAINERS.md](MAINTAINERS.md). Please do not open a public issue for private safety concerns.

Reports will be reviewed as fairly and privately as possible. Maintainers may moderate, edit, remove, or reject contributions that violate this policy.
